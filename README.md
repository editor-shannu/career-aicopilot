# Career AI Co-Pilot

A web application designed to assist users in their career development by analyzing profiles, identifying skill gaps, and providing a personalized action plan.

## Features
- Profile Structuring
- Role Modeling
- Gap Intelligence
- Challenge Planner
- Adaptive Recalibration

## Tech Stack
- Frontend: HTML, CSS, JavaScript (React intended)
- Backend: Node.js/Express (intended)
- AI Integration: GEMINI 

## Setup
1. Clone the repository
2. Install dependencies
3. Set up environment variables
4. Run the application
